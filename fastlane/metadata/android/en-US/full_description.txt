Bearby is a free and open-source multi-chain cryptocurrency wallet. It supports Bitcoin, Ethereum, Binance Smart Chain, and other EVM-compatible networks.

Features:
- Manage multiple accounts and wallets
- Send and receive tokens across supported chains
- Built-in dApp browser with Web3 injection
- QR code scanning for addresses
- Biometric authentication for security
- Hardware wallet support via USB and Bluetooth
- Deep link support for cryptocurrency URLs
- Full control of your private keys — non-custodial

Bearby is built with Flutter and Rust for performance and security. All cryptographic operations run natively via Rust.

This application is fully open source under the GPL-3.0 license.
